# AWS Exam Practice

A study-oriented web application for practicing AWS-style questions with a modern frontend and accessible quiz UI patterns.

**Live demo:** https://aws-exam-practice-bice.vercel.app

## What This Project Shows
- Educational product design around structured practice flows
- Modern frontend component composition for question-heavy interfaces
- Practical use of accessible headless UI primitives

## Why This Stack / Tooling
- **Next.js 15 + React 19** were chosen for a current, deployable app foundation.
- **Radix UI primitives** support accessible dialogs, tabs, radio groups, and progress components for quiz-style interactions.
- **Tailwind-based utility styling** keeps UI iteration fast for a content-heavy educational app.
- **Vercel Speed Insights** helps evaluate frontend performance in a real deployment context.

## Stack
- Next.js 15.5
- React 19
- TypeScript
- Radix UI
- Tailwind CSS

## Key Features
- Exam-practice style question flow
- Accessible component primitives for interaction-heavy screens
- Deployable frontend optimized for fast iteration

## Getting Started
```bash
npm install
npm run dev
```
